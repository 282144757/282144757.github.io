不了解Hexo的小伙伴请戳这里[1]: https://hexo.io “Hexo”,好了，废话不多说，请看正文：

环境安装

1，要使用Hexo必须得安装[2]: https://nodejs.org/en/ “Node.js” 和 Git。
2，在GitHub建立一个新的仓库，名称为 你的GitHub名称+ .github.io,比如我的github名称为282144757，那我就建立一个名称为282144757.github.io的仓库。
3，准备工作做完以后，在本地建立一个目录，名称随意，打开命令行或者终端，然后进入该目录，依次执行
“npm install -g hexo-cli”
“hexo init”
“npm install”
“hexo g”
“hexo s”
如果一路执行下来没有报异常，那此时你应该可以打开浏览器输入“http://localhost:4000”地址，就可以访问到了。

主题更换

在Hexo官网上有很多开源的主题可以免费使用，找到其中一个自己喜欢的，在自己的hexo目录根据主题的使用说明使用“git clone 命令”，把主题clone到本地，然后根据主题使用说明，修改“_config.yml”文件。
然后执行“hexo g”和“hexo s”，再在浏览器中打开“http://localhost:4000”地址，就能看到自己更换的主题了。

总结

其实Hexo使用很简单，大家多在官网看看文档，使用方法介绍的很清楚，我刚开始弄的时候也是一头雾水，最后经过不懈的努力，最终架设成功，看到自己成果，还是有成就感的，话说回来，我们写代码的，不也就这一点乐趣了吗？欢迎大家跟我留言交流！
