这两天做项目要实现一个类似Android通讯录的效果，这里就要用到首字母排序的需求，
在github上找了些案例看了下，大多是通过ascii码实现排序，觉得有点麻烦，于是就看到了一位仁兄提供的一种解决方案，在这里做个记录.


首先通过pinyin4j开源jar包获取到字符串的首字母



    
        public class CnToPinne {
        //获取汉字字符串的第一个字母
        public static String getFirstLetter(String str) {
            StringBuffer sb = new StringBuffer();
            sb.setLength(0);
            char c = str.charAt(0);
            String[] pinyinArray = PinyinHelper.toHanyuPinyinStringArray(c);
            if (pinyinArray != null) {
                sb.append(pinyinArray[0].charAt(0));
            } else {
                sb.append(c);
            }
            return sb.toString();
        }
    }

然后通过自定义Comparator来实现排序。


    public class ContactsComparator implements Comparator{
        private Collator mCollator = Collator.getInstance();
        @Override
        public int compare(ContactsBean o, ContactsBean t1) {
            CollationKey k1=collator.getCollationKey(
                o.getSortChar().toString().toLowerCase());
            CollationKey k2=collator.getCollationKey(
                t1.getSortChar().toString().toLowerCase());
            return k1.compareTo(k2);
        }
    }


最后调用Collections.sort(items, new ContactsComparator());
方法进行集合排序。
